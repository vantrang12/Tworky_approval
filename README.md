# Tworky Approval System

Hệ thống quản lý phiếu đề xuất với Flask và Supabase.

## Tính năng

- Quản lý người dùng
- Quản lý phòng/ban
- Tạo và phê duyệt phiếu đề xuất
- Phân quyền admin/user và approver/normal
